---
title: 'Get Accounting Period'
description: 'Retrieve a specific accounting period'
api: 'GET /service/general-ledger/accounting-periods/{period_id}'
---

## Request

Retrieve details for a specific accounting period.

### Headers

<ParamField header="Authorization" type="string" required>
  JWT authentication token. Format: `jwt YOUR_TOKEN`
</ParamField>

### Path Parameters

<ParamField path="period_id" type="string" required>
  The unique identifier of the accounting period
</ParamField>

<RequestExample>
```bash cURL
curl -X GET https://kube.acho.io/service/general-ledger/accounting-periods/a3UM4R8nYh33A6hgda \
  -H "Authorization: jwt YOUR_TOKEN"
```

```javascript JavaScript
const response = await fetch('https://kube.acho.io/service/general-ledger/accounting-periods/a3UM4R8nYh33A6hgda', {
  method: 'GET',
  headers: {
    'Authorization': 'jwt YOUR_TOKEN'
  }
});

const period = await response.json();
```

```python Python
import requests

response = requests.get(
    'https://kube.acho.io/service/general-ledger/accounting-periods/a3UM4R8nYh33A6hgda',
    headers={
        'Authorization': 'jwt YOUR_TOKEN'
    }
)

period = response.json()
```
</RequestExample>
