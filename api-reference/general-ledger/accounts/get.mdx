---
title: 'Get Account'
description: 'Retrieve a specific GL account by ID'
api: 'GET /service/general-ledger/accounts/{account_id}'
---

## Request

Retrieve details for a specific GL account.

### Headers

<ParamField header="Authorization" type="string" required>
  JWT authentication token. Format: `jwt YOUR_TOKEN`
</ParamField>

### Path Parameters

<ParamField path="account_id" type="string" required>
  The unique identifier of the account to retrieve
</ParamField>

<RequestExample>
```bash cURL
curl -X GET https://kube.acho.io/service/general-ledger/accounts/11100 \
  -H "Authorization: jwt YOUR_TOKEN"
```

```javascript JavaScript
const response = await fetch('https://kube.acho.io/service/general-ledger/accounts/11100', {
  method: 'GET',
  headers: {
    'Authorization': 'jwt YOUR_TOKEN'
  }
});

const account = await response.json();
```

```python Python
import requests

response = requests.get(
    'https://kube.acho.io/service/general-ledger/accounts/11100',
    headers={
        'Authorization': 'jwt YOUR_TOKEN'
    }
)

account = response.json()
```
</RequestExample>
