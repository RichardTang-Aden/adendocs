---

## Request

Retrieve a list of GL accounts with optional filtering by accounting period.

### Headers

<ParamField header="Authorization" type="string" required>
  JWT authentication token. Format: `jwt YOUR_TOKEN`
</ParamField>

### Query Parameters

<ParamField query="accounting_period_id" type="string">
  Filter accounts by accounting period to get period-specific balances
</ParamField>

<ParamField query="account_type" type="string">
  Filter by account type: `Asset`, `Liability`, `Equity`, `Revenue`, `Expense`
</ParamField>

<ParamField query="is_inactive" type="boolean">
  Filter by active status. Set to `true` to include inactive accounts
</ParamField>

<ParamField query="limit" type="number" default="20">
  Maximum number of records to return (max 100)
</ParamField>

<ParamField query="offset" type="number" default="0">
  Number of records to skip for pagination
</ParamField>

<RequestExample>
```bash cURL
curl -X GET "https://kube.acho.io/service/general-ledger/accounts?accounting_period_id=a3UM4R8nYh33A6hgda" \
  -H "Authorization: jwt YOUR_TOKEN"
```

```javascript JavaScript
const params = new URLSearchParams({
  accounting_period_id: 'a3UM4R8nYh33A6hgda'
});

const response = await fetch(`https://kube.acho.io/service/general-ledger/accounts?${params}`, {
  method: 'GET',
  headers: {
    'Authorization': 'jwt YOUR_TOKEN'
  }
});

const accounts = await response.json();
```

```python Python
import requests

response = requests.get(
    'https://kube.acho.io/service/general-ledger/accounts',
    headers={
        'Authorization': 'jwt YOUR_TOKEN'
    },
    params={
        'accounting_period_id': 'a3UM4R8nYh33A6hgda'
    }
)

accounts = response.json()
```
