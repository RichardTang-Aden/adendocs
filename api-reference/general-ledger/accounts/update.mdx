---
title: 'Update Account'
description: 'Update an existing GL account'
api: 'PUT /service/general-ledger/accounts/{account_id}'
---

## Request

Update properties of an existing GL account. Some properties cannot be changed after transactions have been posted to the account.

### Headers

<ParamField header="Authorization" type="string" required>
  JWT authentication token. Format: `jwt YOUR_TOKEN`
</ParamField>

<ParamField header="Content-Type" type="string" required>
  Must be `application/json`
</ParamField>

### Path Parameters

<ParamField path="account_id" type="string" required>
  The unique identifier of the account to update
</ParamField>

### Body Parameters

<ParamField body="account_name" type="string">
  Updated display name for the account
</ParamField>

<ParamField body="is_inactive" type="boolean">
  Set to `true` to deactivate the account
</ParamField>

<Note>
  The following properties cannot be changed after creation:
  - `account_number`
  - `account_type`
  - `normal_balance`
  - `currency`
</Note>

<RequestExample>
```bash cURL
curl -X PUT https://kube.acho.io/service/general-ledger/accounts/11100 \
  -H "Authorization: jwt YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "account_name": "Cash - Main Operating Account",
    "is_inactive": false
  }'
```

```javascript JavaScript
const response = await fetch('https://kube.acho.io/service/general-ledger/accounts/11100', {
  method: 'PUT',
  headers: {
    'Authorization': 'jwt YOUR_TOKEN',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    account_name: 'Cash - Main Operating Account',
    is_inactive: false
  })
});

const account = await response.json();
```

```python Python
import requests

response = requests.put(
    'https://kube.acho.io/service/general-ledger/accounts/11100',
    headers={
        'Authorization': 'jwt YOUR_TOKEN',
        'Content-Type': 'application/json'
    },
    json={
        'account_name': 'Cash - Main Operating Account',
        'is_inactive': False
    }
)

account = response.json()
```
</RequestExample>
