---
title: 'Get Approval Rule'
description: 'Retrieve a specific approval rule'
api: 'GET /service/general-ledger/approval-rules/{rule_id}'
---

## Request

Retrieve details for a specific approval rule.

### Path Parameters

<ParamField path="rule_id" type="string" required>
  The approval rule ID
</ParamField>

<RequestExample>
```bash cURL
curl -X GET https://kube.acho.io/service/general-ledger/approval-rules/APR-BILL-HIGH \
  -H "Authorization: jwt YOUR_TOKEN"
```
</RequestExample>
