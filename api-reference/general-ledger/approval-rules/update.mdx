---
title: 'Update Approval Rule'
description: 'Update an existing approval rule'
api: 'PUT /service/general-ledger/approval-rules/{rule_id}'
---

## Request

Update an existing approval rule.

### Path Parameters

<ParamField path="rule_id" type="string" required>
  The approval rule ID
</ParamField>

### Body Parameters

<ParamField body="min_amount" type="number">
  Minimum amount threshold
</ParamField>

<ParamField body="max_amount" type="number">
  Maximum amount threshold
</ParamField>

<ParamField body="is_active" type="boolean">
  Whether the rule is active
</ParamField>

<ParamField body="priority" type="integer">
  Priority order
</ParamField>

<ParamField body="rule_name" type="string">
  Human-readable name
</ParamField>

<ParamField body="approval_level" type="integer">
  Required approval level
</ParamField>

<ParamField body="approver_type" type="string">
  Approver type: `role`, `user`, `manager`
</ParamField>

<ParamField body="approver_role" type="string">
  Role required for approval
</ParamField>

<RequestExample>
```bash cURL
curl -X PUT https://kube.acho.io/service/general-ledger/approval-rules/APR-BILL-HIGH \
  -H "Authorization: jwt YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "min_amount": 15000,
    "is_active": true,
    "priority": 20
  }'
```
</RequestExample>
