---
title: 'Clear Bill Payment'
description: 'Mark a bill payment as cleared (bank reconciliation)'
api: 'POST /service/general-ledger/bill-payments/{bill_payment_id}/clear'
---

## Request

Mark a bill payment as cleared for bank reconciliation purposes.

### Headers

<ParamField header="Authorization" type="string" required>
  JWT authentication token. Format: `jwt YOUR_TOKEN`
</ParamField>

<ParamField header="Content-Type" type="string" required>
  Must be `application/json`
</ParamField>

### Path Parameters

<ParamField path="bill_payment_id" type="string" required>
  The payment ID
</ParamField>

### Body Parameters

<ParamField body="cleared_date" type="string" required>
  Date the payment cleared the bank (YYYY-MM-DD)
</ParamField>

<RequestExample>
```bash cURL
curl -X POST https://kube.acho.io/service/general-ledger/bill-payments/BP-2024-001/clear \
  -H "Authorization: jwt YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"cleared_date": "2024-02-12"}'
```
</RequestExample>

<ResponseExample>
```json 200 OK
{
  "bill_payment_id": "BP-2024-001",
  "status": "Cleared",
  "cleared_date": "2024-02-12",
  "cleared_at": "2024-02-15T10:00:00Z"
}
```
</ResponseExample>
