title: 'Create Bill Payment'
description: 'Create a new payment to a vendor'
api: 'POST /service/general-ledger/bill-payments'
---

## Request

Create a new bill payment to pay one or more vendor bills.

### Headers

<ParamField header="Authorization" type="string" required>
  JWT authentication token. Format: `jwt YOUR_TOKEN`
</ParamField>

<ParamField header="Content-Type" type="string" required>
  Must be `application/json`
</ParamField>

### Body Parameters

<ParamField body="bill_payment_id" type="string" required>
  Unique identifier for the payment
</ParamField>

<ParamField body="vendor_id" type="string" required>
  Vendor being paid
</ParamField>

<ParamField body="account_id" type="string" required>
  Cash/bank account for the payment
</ParamField>

<ParamField body="ap_account_id" type="string" required>
  Accounts Payable GL account
</ParamField>

<ParamField body="tran_date" type="string" required>
  Payment date (YYYY-MM-DD)
</ParamField>

<ParamField body="posting_period_id" type="string" required>
  Accounting period ID
</ParamField>

<ParamField body="total" type="number" required>
  Total payment amount
</ParamField>

<ParamField body="currency" type="string" default="USD">
  Payment currency
</ParamField>

<ParamField body="exchange_rate" type="number" default="1.0">
  Exchange rate for multi-currency
</ParamField>

<ParamField body="memo" type="string">
  Payment memo/description
</ParamField>

<ParamField body="status" type="string" default="Draft">
  Initial status
</ParamField>

<RequestExample>
```bash cURL
curl -X POST https://kube.acho.io/service/general-ledger/bill-payments \
  -H "Authorization: jwt YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "bill_payment_id": "BP-2024-001",
    "vendor_id": "a3-vendor-xxx",
    "account_id": "a3-cash-account-xxx",
    "ap_account_id": "a3-ap-account-xxx",
    "tran_date": "2024-02-10",
    "posting_period_id": "a3-period-xxx",
    "total": 1000,
    "currency": "USD",
    "exchange_rate": 1.0,
    "memo": "Payment for BILL-2024-001",
    "status": "Draft"
  }'
```
</RequestExample>
