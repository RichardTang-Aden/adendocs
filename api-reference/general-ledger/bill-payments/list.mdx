## Request

Retrieve a list of bill payments with optional filtering.

### Headers

<ParamField header="Authorization" type="string" required>
  JWT authentication token. Format: `jwt YOUR_TOKEN`
</ParamField>

### Query Parameters

<ParamField query="status" type="string">
  Filter by status: `Draft`, `Pending Approval`, `Approved`, `On Hold`, `Rejected`, `Voided`, `Pending`, `Cleared`
</ParamField>

<ParamField query="vendor_id" type="string">
  Filter by vendor ID
</ParamField>

<ParamField query="start_date" type="string">
  Filter payments on or after date (YYYY-MM-DD)
</ParamField>

<ParamField query="end_date" type="string">
  Filter payments on or before date (YYYY-MM-DD)
</ParamField>

<RequestExample>
```bash cURL
curl -X GET "https://kube.acho.io/service/general-ledger/bill-payments?status=Draft&start_date=2024-01-01&end_date=2024-12-31" \
  -H "Authorization: jwt YOUR_TOKEN"
```
</RequestExample>
