---
title: 'Void Bill Payment'
description: 'Void a bill payment'
api: 'POST /service/general-ledger/bill-payments/{bill_payment_id}/void'
---

## Request

Void a bill payment. This reverses the payment's effect on the bill balance.

### Headers

<ParamField header="Authorization" type="string" required>
  JWT authentication token. Format: `jwt YOUR_TOKEN`
</ParamField>

<ParamField header="Content-Type" type="string" required>
  Must be `application/json`
</ParamField>

### Path Parameters

<ParamField path="bill_payment_id" type="string" required>
  The payment ID
</ParamField>

### Body Parameters

<ParamField body="reason" type="string" required>
  Reason for voiding
</ParamField>

<RequestExample>
```bash cURL
curl -X POST https://kube.acho.io/service/general-ledger/bill-payments/BP-2024-001/void \
  -H "Authorization: jwt YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"reason": "Payment cancelled"}'
```
</RequestExample>

<ResponseExample>
```json 200 OK
{
  "bill_payment_id": "BP-2024-001",
  "status": "Voided",
  "void_reason": "Payment cancelled",
  "voided_at": "2024-02-15T10:00:00Z"
}
```
</ResponseExample>
