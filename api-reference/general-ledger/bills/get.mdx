---
title: 'Get Bill'
description: 'Retrieve a specific vendor bill'
api: 'GET /service/general-ledger/bills/{bill_id}'
---

## Request

Retrieve details for a specific vendor bill.

### Headers

<ParamField header="Authorization" type="string" required>
  JWT authentication token. Format: `jwt YOUR_TOKEN`
</ParamField>

### Path Parameters

<ParamField path="bill_id" type="string" required>
  The unique identifier of the bill
</ParamField>

<RequestExample>
```bash cURL
curl -X GET https://kube.acho.io/service/general-ledger/bills/BILL-2024-001 \
  -H "Authorization: jwt YOUR_TOKEN"
```

```javascript JavaScript
const response = await fetch('https://kube.acho.io/service/general-ledger/bills/BILL-2024-001', {
  method: 'GET',
  headers: {
    'Authorization': 'jwt YOUR_TOKEN'
  }
});

const bill = await response.json();
```

```python Python
import requests

response = requests.get(
    'https://kube.acho.io/service/general-ledger/bills/BILL-2024-001',
    headers={
        'Authorization': 'jwt YOUR_TOKEN'
    }
)

bill = response.json()
```
</RequestExample>
