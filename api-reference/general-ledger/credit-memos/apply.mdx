---
title: 'Apply Credit Memo'
description: 'Apply a credit memo to an invoice'
api: 'POST /service/general-ledger/credit-memos/{credit_memo_id}/apply'
---

## Request

Apply an approved credit memo to an outstanding invoice.

### Path Parameters

<ParamField path="credit_memo_id" type="string" required>
  The credit memo ID
</ParamField>

### Body Parameters

<ParamField body="invoice_id" type="string" required>
  Invoice to apply credit to
</ParamField>

<ParamField body="amount" type="number" required>
  Amount to apply
</ParamField>

<RequestExample>
```bash cURL
curl -X POST https://kube.acho.io/service/general-ledger/credit-memos/CM-2024-001/apply \
  -H "Authorization: jwt YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "invoice_id": "a3-invoice-xxx",
    "amount": 500
  }'
```
</RequestExample>

<ResponseExample>
```json 200 OK
{
  "credit_memo_id": "CM-2024-001",
  "invoice_id": "a3-invoice-xxx",
  "amount_applied": 500,
  "credit_memo_status": "Applied",
  "credit_memo_remaining": 0,
  "invoice_amount_due": 3900
}
```
</ResponseExample>
