## Request

Create a new credit memo for a customer.

### Body Parameters

<ParamField body="credit_memo_id" type="string" required>
  Unique identifier
</ParamField>

<ParamField body="customer_id" type="string" required>
  Customer ID
</ParamField>

<ParamField body="ar_account_id" type="string" required>
  Accounts Receivable account
</ParamField>

<ParamField body="tran_date" type="string" required>
  Credit memo date (YYYY-MM-DD)
</ParamField>

<ParamField body="posting_period_id" type="string" required>
  Accounting period
</ParamField>

<ParamField body="total" type="number" required>
  Credit amount
</ParamField>

<ParamField body="memo" type="string">
  Description/reason
</ParamField>

<ParamField body="lines" type="array" required>
  Line items with revenue accounts
</ParamField>
