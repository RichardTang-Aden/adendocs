## Request

Retrieve details for a specific credit memo.

### Path Parameters

<ParamField path="credit_memo_id" type="string" required>
  The credit memo ID
</ParamField>

<RequestExample>
```bash cURL
curl -X GET https://kube.acho.io/service/general-ledger/credit-memos/CM-2024-001 \
  -H "Authorization: jwt YOUR_TOKEN"
```
</RequestExample>
