## Overview

Credit Memos reduce amounts owed by customers. They can be issued for returns, billing adjustments, or service credits, and can be applied against outstanding invoices.

## Credit Memo Statuses

| Status | Description |
|--------|-------------|
| `Draft` | Initial state |
| `Pending Approval` | Submitted for approval |
| `On Hold` | Temporarily held |
| `Approved` | Approved |
| `Rejected` | Rejected |
| `Voided` | Voided |
| `Partially Applied` | Partially applied to invoices |
| `Applied` | Fully applied |

## Accounting Impact

When a credit memo is posted:
- **Debit**: Revenue account (reduces revenue)
- **Credit**: Accounts Receivable (reduces customer balance)
