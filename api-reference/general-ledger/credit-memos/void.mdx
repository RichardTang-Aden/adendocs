---
title: 'Void Credit Memo'
description: 'Void a credit memo'
api: 'POST /service/general-ledger/credit-memos/{credit_memo_id}/void'
---

## Request

Void a credit memo.

### Body Parameters

<ParamField body="reason" type="string" required>
  Reason for voiding
</ParamField>

<Warning>
  Applied credit memos cannot be voided. You must unapply them first.
</Warning>

<RequestExample>
```bash cURL
curl -X POST https://kube.acho.io/service/general-ledger/credit-memos/CM-2024-001/void \
  -H "Authorization: jwt YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"reason": "Credit memo created in error"}'
```
</RequestExample>

<ResponseExample>
```json 200 OK
{
  "credit_memo_id": "CM-2024-001",
  "status": "Voided",
  "void_reason": "Credit memo created in error"
}
```
</ResponseExample>
