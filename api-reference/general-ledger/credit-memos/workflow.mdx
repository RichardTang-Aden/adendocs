---

## Workflow Actions

### Submit for Approval
```bash
POST /service/general-ledger/credit-memos/{credit_memo_id}/submit-for-approval
```

### Approve
```bash
POST /service/general-ledger/credit-memos/{credit_memo_id}/approve
```

### Reject
```bash
POST /service/general-ledger/credit-memos/{credit_memo_id}/reject
```
Body: `{"reason": "..."}`

### Request Change
```bash
POST /service/general-ledger/credit-memos/{credit_memo_id}/request-change
```
Body: `{"comments": "..."}`

### Place on Hold
```bash
POST /service/general-ledger/credit-memos/{credit_memo_id}/hold
```
Body: `{"reason": "..."}`

### Remove from Hold
```bash
POST /service/general-ledger/credit-memos/{credit_memo_id}/unhold
```

### Get Audit Trail
```bash
GET /service/general-ledger/credit-memos/{credit_memo_id}/audit-trail
```
