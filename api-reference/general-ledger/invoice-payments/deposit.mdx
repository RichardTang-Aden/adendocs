---
title: 'Deposit Invoice Payment'
description: 'Mark an invoice payment as deposited'
api: 'POST /service/general-ledger/invoice-payments/{invoice_payment_id}/deposit'
---

## Request

Mark an invoice payment as deposited to the bank.

### Body Parameters

<ParamField body="deposit_date" type="string" required>
  Date of deposit (YYYY-MM-DD)
</ParamField>

<RequestExample>
```bash cURL
curl -X POST https://kube.acho.io/service/general-ledger/invoice-payments/CP-2024-001/deposit \
  -H "Authorization: jwt YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"deposit_date": "2024-02-12"}'
```
</RequestExample>

<ResponseExample>
```json 200 OK
{
  "customer_payment_id": "CP-2024-001",
  "status": "Deposited",
  "deposit_date": "2024-02-12"
}
```
</ResponseExample>
