## Request

Retrieve details for a specific invoice payment.

### Path Parameters

<ParamField path="invoice_payment_id" type="string" required>
  The payment ID
</ParamField>

<RequestExample>
```bash cURL
curl -X GET https://kube.acho.io/service/general-ledger/invoice-payments/CP-2024-001 \
  -H "Authorization: jwt YOUR_TOKEN"
```
</RequestExample>
