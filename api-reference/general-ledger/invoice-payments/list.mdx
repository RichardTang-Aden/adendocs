## Request

Retrieve a list of invoice payments.

### Query Parameters

<ParamField query="status" type="string">
  Filter by status: `Draft`, `Pending Approval`, `Approved`, `On Hold`, `Rejected`, `Voided`, `Pending`, `Deposited`
</ParamField>

<ParamField query="customer_id" type="string">
  Filter by customer
</ParamField>

<ParamField query="start_date" type="string">
  Filter payments on or after date
</ParamField>

<ParamField query="end_date" type="string">
  Filter payments on or before date
</ParamField>

<RequestExample>
```bash cURL
curl -X GET "https://kube.acho.io/service/general-ledger/invoice-payments?status=Draft&start_date=2024-01-01&end_date=2024-12-31" \
  -H "Authorization: jwt YOUR_TOKEN"
```
</RequestExample>
