## Overview

The Invoice Payments API manages customer payments received for outstanding invoices. Payments can be applied to single or multiple invoices.

## Payment Statuses

| Status | Description |
|--------|-------------|
| `Draft` | Initial state |
| `Pending Approval` | Submitted for approval |
| `On Hold` | Temporarily held |
| `Approved` | Approved |
| `Rejected` | Rejected |
| `Voided` | Voided |
| `Pending` | Awaiting deposit |
| `Deposited` | Deposited to bank |
