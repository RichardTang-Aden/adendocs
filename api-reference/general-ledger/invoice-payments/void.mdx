---
title: 'Void Invoice Payment'
description: 'Void an invoice payment'
api: 'POST /service/general-ledger/invoice-payments/{invoice_payment_id}/void'
---

## Request

Void an invoice payment.

### Body Parameters

<ParamField body="reason" type="string" required>
  Reason for voiding
</ParamField>

<RequestExample>
```bash cURL
curl -X POST https://kube.acho.io/service/general-ledger/invoice-payments/CP-2024-001/void \
  -H "Authorization: jwt YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"reason": "Check bounced"}'
```
</RequestExample>

<ResponseExample>
```json 200 OK
{
  "customer_payment_id": "CP-2024-001",
  "status": "Voided",
  "void_reason": "Check bounced"
}
```
</ResponseExample>
