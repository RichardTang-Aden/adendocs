description: 'Retrieve a specific customer invoice'
api: 'GET /service/general-ledger/invoices/{invoice_id}'
---

## Request

Retrieve details for a specific customer invoice.

### Headers

<ParamField header="Authorization" type="string" required>
  JWT authentication token. Format: `jwt YOUR_TOKEN`
</ParamField>

### Path Parameters

<ParamField path="invoice_id" type="string" required>
  The unique identifier of the invoice
</ParamField>

<RequestExample>
```bash cURL
curl -X GET https://kube.acho.io/service/general-ledger/invoices/INV-2024-001 \
  -H "Authorization: jwt YOUR_TOKEN"
```
</RequestExample>
