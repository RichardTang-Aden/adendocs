---
title: 'Receive Payment'
description: 'Record a customer payment against an invoice'
api: 'POST /service/general-ledger/invoices/{invoice_id}/receive-payment'
---

## Request

Record a customer payment against an invoice.

### Headers

<ParamField header="Authorization" type="string" required>
  JWT authentication token. Format: `jwt YOUR_TOKEN`
</ParamField>

<ParamField header="Content-Type" type="string" required>
  Must be `application/json`
</ParamField>

### Path Parameters

<ParamField path="invoice_id" type="string" required>
  The invoice ID
</ParamField>

### Body Parameters

<ParamField body="payment_date" type="string" required>
  Date of payment (YYYY-MM-DD)
</ParamField>

<ParamField body="payment_amount" type="number" required>
  Amount received
</ParamField>

<ParamField body="deposit_account_id" type="string" required>
  Cash/bank account for the deposit
</ParamField>

<ParamField body="payment_method" type="string">
  Payment method (e.g., `Bank Transfer`, `Check`, `Credit Card`)
</ParamField>

<ParamField body="reference" type="string">
  Payment reference number
</ParamField>

<RequestExample>
```bash cURL
curl -X POST https://kube.acho.io/service/general-ledger/invoices/INV-2024-001/receive-payment \
  -H "Authorization: jwt YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "payment_date": "2024-02-10",
    "payment_amount": 4400,
    "deposit_account_id": "a3-cash-xxx",
    "payment_method": "Bank Transfer",
    "reference": "WIRE-001"
  }'
```
</RequestExample>

<ResponseExample>
```json 200 OK
{
  "invoice_id": "INV-2024-001",
  "payment_id": "CP-2024-001",
  "payment_date": "2024-02-10",
  "payment_amount": 4400,
  "deposit_account_id": "a3-cash-xxx",
  "payment_method": "Bank Transfer",
  "reference": "WIRE-001",
  "invoice_status": "Paid",
  "amount_paid_total": 4400,
  "amount_due": 0
}
```
</ResponseExample>
