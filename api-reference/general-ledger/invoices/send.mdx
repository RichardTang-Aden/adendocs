---
title: 'Send Invoice'
description: 'Send an approved invoice to the customer'
api: 'POST /service/general-ledger/invoices/{invoice_id}/send'
---

## Request

Send an approved invoice to the customer. Status changes to "Sent".

### Headers

<ParamField header="Authorization" type="string" required>
  JWT authentication token. Format: `jwt YOUR_TOKEN`
</ParamField>

### Path Parameters

<ParamField path="invoice_id" type="string" required>
  The invoice ID
</ParamField>

<RequestExample>
```bash cURL
curl -X POST https://kube.acho.io/service/general-ledger/invoices/INV-2024-001/send \
  -H "Authorization: jwt YOUR_TOKEN"
```
</RequestExample>

<ResponseExample>
```json 200 OK
{
  "invoice_id": "INV-2024-001",
  "status": "Sent",
  "sent_at": "2024-01-16T10:00:00Z",
  "sent_to": "billing@acmecorp.com"
}
```
</ResponseExample>
