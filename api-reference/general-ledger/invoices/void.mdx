---
title: 'Void Invoice'
description: 'Void a customer invoice'
api: 'POST /service/general-ledger/invoices/{invoice_id}/void'
---

## Request

Void a customer invoice.

### Headers

<ParamField header="Authorization" type="string" required>
  JWT authentication token. Format: `jwt YOUR_TOKEN`
</ParamField>

<ParamField header="Content-Type" type="string" required>
  Must be `application/json`
</ParamField>

### Path Parameters

<ParamField path="invoice_id" type="string" required>
  The invoice ID
</ParamField>

### Body Parameters

<ParamField body="reason" type="string" required>
  Reason for voiding
</ParamField>

<Warning>
  Invoices with payments cannot be voided. You must first void any associated payments or issue a credit memo.
</Warning>

<RequestExample>
```bash cURL
curl -X POST https://kube.acho.io/service/general-ledger/invoices/INV-2024-001/void \
  -H "Authorization: jwt YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"reason": "Customer cancelled order"}'
```
</RequestExample>

<ResponseExample>
```json 200 OK
{
  "invoice_id": "INV-2024-001",
  "status": "Voided",
  "void_reason": "Customer cancelled order",
  "voided_at": "2024-01-20T10:00:00Z"
}
```
</ResponseExample>
