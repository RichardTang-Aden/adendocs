---
title: 'Delete Journal Entry'
description: 'Delete a draft journal entry'
api: 'DELETE /service/general-ledger/journal-entries/{journal_entry_id}'
---

## Request

Permanently delete a journal entry. Only entries in `Draft` status can be deleted.

<Warning>
  Deletion is permanent and cannot be undone. For entries that have been submitted or posted, use the Void action instead.
</Warning>

### Headers

<ParamField header="Authorization" type="string" required>
  JWT authentication token. Format: `jwt YOUR_TOKEN`
</ParamField>

### Path Parameters

<ParamField path="journal_entry_id" type="string" required>
  The unique identifier of the journal entry to delete
</ParamField>

<RequestExample>
```bash cURL
curl -X DELETE https://kube.acho.io/service/general-ledger/journal-entries/JE-2024-001 \
  -H "Authorization: jwt YOUR_TOKEN"
```

```javascript JavaScript
const response = await fetch('https://kube.acho.io/service/general-ledger/journal-entries/JE-2024-001', {
  method: 'DELETE',
  headers: {
    'Authorization': 'jwt YOUR_TOKEN'
  }
});

if (response.status === 204) {
  console.log('Entry deleted successfully');
}
```

```python Python
import requests

response = requests.delete(
    'https://kube.acho.io/service/general-ledger/journal-entries/JE-2024-001',
    headers={
        'Authorization': 'jwt YOUR_TOKEN'
    }
)

if response.status_code == 204:
    print('Entry deleted successfully')
```
</RequestExample>

## When to Delete vs. Void

| Status | Can Delete? | Alternative |
|--------|-------------|-------------|
| `Draft` | Yes | - |
| `Pending Approval` | No | Request Change to return to Draft, then delete |
| `On Hold` | No | Request Change to return to Draft, then delete |
| `Approved` | No | Void the entry |
| `Posted` | No | Void or Reverse the entry |
| `Rejected` | No | Void the entry |
| `Voided` | No | Already voided |
