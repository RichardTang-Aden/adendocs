---
title: 'Get Journal Entry'
description: 'Retrieve a specific journal entry'
api: 'GET /service/general-ledger/journal-entries/{journal_entry_id}'
---

## Request

Retrieve details for a specific journal entry.

### Headers

<ParamField header="Authorization" type="string" required>
  JWT authentication token. Format: `jwt YOUR_TOKEN`
</ParamField>

### Path Parameters

<ParamField path="journal_entry_id" type="string" required>
  The unique identifier of the journal entry
</ParamField>

<RequestExample>
```bash cURL
curl -X GET https://kube.acho.io/service/general-ledger/journal-entries/JE-2024-002 \
  -H "Authorization: jwt YOUR_TOKEN"
```

```javascript JavaScript
const response = await fetch('https://kube.acho.io/service/general-ledger/journal-entries/JE-2024-002', {
  method: 'GET',
  headers: {
    'Authorization': 'jwt YOUR_TOKEN'
  }
});

const entry = await response.json();
```

```python Python
import requests

response = requests.get(
    'https://kube.acho.io/service/general-ledger/journal-entries/JE-2024-002',
    headers={
        'Authorization': 'jwt YOUR_TOKEN'
    }
)

entry = response.json()
```
</RequestExample>
