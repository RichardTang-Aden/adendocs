---
title: 'Clear Transactions'
description: 'Mark transactions as cleared in reconciliation'
api: 'POST /service/general-ledger/reconciliations/{reconciliation_id}/clear'
---

## Request

Mark bank transactions as cleared in this reconciliation session.

### Path Parameters

<ParamField path="reconciliation_id" type="string" required>
  The reconciliation session ID
</ParamField>

### Body Parameters

<ParamField body="transactionIds" type="array" required>
  Array of transaction IDs to clear
</ParamField>

<ParamField body="clearedDate" type="string" required>
  Date the transactions cleared (YYYY-MM-DD)
</ParamField>

<RequestExample>
```bash cURL
curl -X POST https://kube.acho.io/service/general-ledger/reconciliations/recon-2024-001/clear \
  -H "Authorization: jwt YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "transactionIds": ["txn_id_1", "txn_id_2", "txn_id_3"],
    "clearedDate": "2024-01-31"
  }'
```
</RequestExample>

<ResponseExample>
```json 200 OK
{
  "reconciliation_id": "recon-2024-001",
  "transactions_cleared": 3,
  "cleared_balance": 53000.00,
  "uncleared_balance": -1000.00,
  "difference": 0
}
```
</ResponseExample>
