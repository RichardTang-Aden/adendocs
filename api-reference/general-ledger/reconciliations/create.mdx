---
title: 'Create Reconciliation Session'
description: 'Create a new bank reconciliation session'
api: 'POST /service/general-ledger/reconciliations'
---

## Request

Create a new bank reconciliation session. The GL ending balance is automatically calculated based on the linked ledger account.

### Body Parameters

<ParamField body="bankAccountId" type="string" required>
  The bank account ID to reconcile
</ParamField>

<ParamField body="statementDate" type="string" required>
  Bank statement date (YYYY-MM-DD)
</ParamField>

<ParamField body="statementEndingBalance" type="number" required>
  Ending balance from bank statement
</ParamField>

<ParamField body="statementBeginningBalance" type="number" required>
  Beginning balance from bank statement
</ParamField>

<ParamField body="postingPeriodId" type="string" required>
  Accounting period for this reconciliation
</ParamField>

<ParamField body="currency" type="string" required>
  Currency code (e.g., `USD`)
</ParamField>

<ParamField body="memo" type="string">
  Optional description for this reconciliation
</ParamField>

<RequestExample>
```bash cURL
curl -X POST https://kube.acho.io/service/general-ledger/reconciliations \
  -H "Authorization: jwt YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "bankAccountId": "a3-bank-account-xxx",
    "statementDate": "2024-01-31",
    "statementEndingBalance": 50000.00,
    "statementBeginningBalance": 45000.00,
    "postingPeriodId": "a3-period-xxx",
    "currency": "USD",
    "memo": "January 2024 Bank Reconciliation"
  }'
```
</RequestExample>
