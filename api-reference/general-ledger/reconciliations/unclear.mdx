---
title: 'Unclear Transactions'
description: 'Remove transactions from cleared list'
api: 'POST /service/general-ledger/reconciliations/{reconciliation_id}/unclear'
---

## Request

Remove transactions from the cleared list in this reconciliation session.

### Path Parameters

<ParamField path="reconciliation_id" type="string" required>
  The reconciliation session ID
</ParamField>

### Body Parameters

<ParamField body="transactionIds" type="array" required>
  Array of transaction IDs to unclear
</ParamField>

<RequestExample>
```bash cURL
curl -X POST https://kube.acho.io/service/general-ledger/reconciliations/recon-2024-001/unclear \
  -H "Authorization: jwt YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "transactionIds": ["txn_id_1", "txn_id_2"]
  }'
```
</RequestExample>

<ResponseExample>
```json 200 OK
{
  "reconciliation_id": "recon-2024-001",
  "transactions_uncleared": 2,
  "cleared_balance": 48000.00,
  "uncleared_balance": 4000.00,
  "difference": 2000.00
}
```
</ResponseExample>
