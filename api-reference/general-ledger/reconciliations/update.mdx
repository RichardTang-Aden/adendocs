---
title: 'Update Reconciliation Session'
description: 'Update reconciliation session details'
api: 'PUT /service/general-ledger/reconciliations/{reconciliation_id}'
---

## Request

Update reconciliation session details including adjustments.

### Path Parameters

<ParamField path="reconciliation_id" type="string" required>
  The reconciliation session ID
</ParamField>

### Body Parameters

<ParamField body="statementEndingBalance" type="number">
  Updated ending balance from bank statement
</ParamField>

<ParamField body="bankAdjustments" type="number">
  Bank adjustment amount
</ParamField>

<ParamField body="bookAdjustments" type="number">
  Book adjustment amount
</ParamField>

<ParamField body="memo" type="string">
  Updated memo
</ParamField>

<RequestExample>
```bash cURL
curl -X PUT https://kube.acho.io/service/general-ledger/reconciliations/recon-2024-001 \
  -H "Authorization: jwt YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "statementEndingBalance": 50500.00,
    "bankAdjustments": 100.00,
    "bookAdjustments": -50.00,
    "memo": "Updated reconciliation memo"
  }'
```
</RequestExample>

<ResponseExample>
```json 200 OK
{
  "reconciliation_id": "recon-2024-001",
  "statement_ending_balance": 50500.00,
  "bank_adjustments": 100.00,
  "book_adjustments": -50.00,
  "memo": "Updated reconciliation memo",
  "updated_at": "2024-02-01T11:00:00Z",
  "updated_by": "user-123"
}
```
</ResponseExample>
