title: 'Get Reconciliation Worksheet'
description: 'Get the reconciliation worksheet with transactions'
api: 'GET /service/general-ledger/reconciliations/{reconciliation_id}/worksheet'
---

## Request

Get the reconciliation worksheet with cleared/uncleared transactions and balance summary.

### Path Parameters

<ParamField path="reconciliation_id" type="string" required>
  The reconciliation session ID
</ParamField>

<RequestExample>
```bash cURL
curl -X GET https://kube.acho.io/service/general-ledger/reconciliations/recon-2024-001/worksheet \
  -H "Authorization: jwt YOUR_TOKEN"
```
</RequestExample>
