---

## Request

Generate a balance sheet showing assets, liabilities, and equity as of a specific date.

### Query Parameters

<ParamField query="as_of_date" type="string" required>
  Date for the balance sheet (YYYY-MM-DD)
</ParamField>

<ParamField query="posting_period_id" type="string">
  Alternative: Filter by accounting period
</ParamField>

<RequestExample>
```bash cURL
curl -X GET "https://kube.acho.io/service/general-ledger/reports/balance-sheet?as_of_date=2024-01-31" \
  -H "Authorization: jwt YOUR_TOKEN"
```
</RequestExample>
