## Request

Generate a cash flow statement showing cash movements for a date range.

### Query Parameters

<ParamField query="start_date" type="string" required>
  Start date (YYYY-MM-DD)
</ParamField>

<ParamField query="end_date" type="string" required>
  End date (YYYY-MM-DD)
</ParamField>

<ParamField query="posting_period_id" type="string">
  Alternative: Filter by accounting period
</ParamField>

<RequestExample>
```bash cURL
curl -X GET "https://kube.acho.io/service/general-ledger/reports/cash-flow?start_date=2024-01-01&end_date=2024-01-31" \
  -H "Authorization: jwt YOUR_TOKEN"
```
</RequestExample>
