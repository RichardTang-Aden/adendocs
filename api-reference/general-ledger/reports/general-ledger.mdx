---

## Request

Generate a detailed general ledger report showing all transactions for an account.

### Query Parameters

<ParamField query="account_id" type="string" required>
  The account ID to report on
</ParamField>

<ParamField query="posting_period_id" type="string" required>
  The accounting period ID
</ParamField>

<RequestExample>
```bash cURL
curl -X GET "https://kube.acho.io/service/general-ledger/reports/general-ledger?account_id=1000&posting_period_id=a3-period-xxx" \
  -H "Authorization: jwt YOUR_TOKEN"
```
</RequestExample>
