title: 'Income Statement'

api: 'GET /service/general-ledger/reports/income-statement'
---

## Request

Generate an income statement (profit and loss) report for a date range.

### Query Parameters

<ParamField query="start_date" type="string" required>
  Start date (YYYY-MM-DD)
</ParamField>

<ParamField query="end_date" type="string" required>
  End date (YYYY-MM-DD)
</ParamField>

<ParamField query="posting_period_id" type="string">
  Alternative: Filter by accounting period
</ParamField>

<RequestExample>
```bash cURL
curl -X GET "https://kube.acho.io/service/general-ledger/reports/income-statement?start_date=2024-01-01&end_date=2024-01-31" \
  -H "Authorization: jwt YOUR_TOKEN"
```
</RequestExample>
