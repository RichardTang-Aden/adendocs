## Overview

The Reports API provides access to standard financial reports generated from your General Ledger data.

## Available Reports

### Financial Statements

| Report | Description |
|--------|-------------|
| [Trial Balance](/api-reference/general-ledger/reports/trial-balance) | Account balances for a period |
| [General Ledger](/api-reference/general-ledger/reports/general-ledger) | Transaction detail by account |
| [Balance Sheet](/api-reference/general-ledger/reports/balance-sheet) | Assets, liabilities, equity as of date |
| [Income Statement](/api-reference/general-ledger/reports/income-statement) | Revenue and expenses for period |
| [Cash Flow Statement](/api-reference/general-ledger/reports/cash-flow) | Cash movements for period |

### Operational Reports

| Report | Description |
|--------|-------------|
| [Pending Approvals](/api-reference/general-ledger/reports/pending-approvals) | Documents awaiting approval |
| [AP Aging](/api-reference/general-ledger/reports/ap-aging) | Accounts payable aging by due date |
| [AR Aging](/api-reference/general-ledger/reports/ar-aging) | Accounts receivable aging by due date |

## Common Parameters

Most reports support these common parameters:

| Parameter | Description |
|-----------|-------------|
| `posting_period_id` | Filter by accounting period |
| `start_date` | Beginning of date range |
| `end_date` | End of date range |
| `as_of_date` | Point-in-time snapshot date |

## Report Format

All reports return JSON data that can be consumed directly or transformed for presentation.
