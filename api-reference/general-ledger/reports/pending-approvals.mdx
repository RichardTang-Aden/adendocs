title: 'Pending Approvals Report'
description: 'Get documents pending approval'
api: 'GET /service/general-ledger/reports/pending-approvals'
---

## Request

Returns all documents pending approval across all document types, filtered by the current user's IAM permissions.

### Query Parameters

<ParamField query="document_type" type="string">
  Filter by document type: `all`, `journal_entry`, `bill`, `invoice`, `bill_payment`, `invoice_payment`, `credit_memo`. Default: `all`
</ParamField>

<ParamField query="approval_level" type="integer">
  Filter by approval level (1, 2, 3, etc.)
</ParamField>

<RequestExample>
```bash cURL
curl -X GET "https://kube.acho.io/service/general-ledger/reports/pending-approvals?document_type=all" \
  -H "Authorization: jwt YOUR_TOKEN"
```
</RequestExample>
